一个简单的包裹登记程序，使用微信小程序做前端，python FASTAPI做后端，python tkinter做管理端

微信小程序端使用JavaScript，可注册账号密码OR调用微信OPENID自动创建账号一键登录,使用微信条目扫描接口实现使用摄像头/相册图片自动识别输入快递单号
